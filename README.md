# EAMS

electronic album management system

## 项目介绍
本项目是软件工程课程的大作业，是一个电子相册管理系统，主要功能有：
1. 用户注册、登录、注销
2. 用户上传图片
3. 用户管理相册
4. 用户管理图片
5. 用户管理个人信息
6. 用户管理好友
